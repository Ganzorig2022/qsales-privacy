 # qsales-privacy

  Public privacy policy for the **QSales** Android app (`mn.qpay.qsales`) — QPay's
  merchant onboarding / sales-agent tool.

  This repository exists solely to host the privacy policy at a public HTTPS URL,
  as required by the Google Play Console (**App content → Privacy policy**).

  ## Live policy

  📄 **https://ganzorig2022.github.io/qsales-privacy/**

  Served via GitHub Pages from [`index.md`](./index.md) on the `main` branch.

  ## Maintenance

  - The canonical source lives in the app repo at
    `docs/play-readiness/PRIVACY_POLICY_FINAL.md`.
  - To update: edit `index.md` here, commit to `main`, and GitHub Pages
    redeploys automatically (~1–2 min). Keep both files in sync.
  - Bump the **Last updated** date in `index.md` whenever the policy changes.
